# Heat.js - Binding Options:

Below is a list of all the options supported in the "data-heat-options" binding attribute for DOM elements.
<br>
<br>


## Standard Options:

| Type: | Name: | Description: |
| --- | --- | --- |
| *number* | timeout | States the delay (in milliseconds) that should be used before checking the object for changes (defaults to 250 milliseconds). |

<br/>


## Binding Example:

```markdown
<div data-heat-options="{  }">
    Your HTML.
</div>
```
